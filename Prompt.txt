🦷 Professional Prompt: SmileCare Dental — Premium Dental Clinic Website
Project Goal: Design a high-conversion, trust-building website for SmileCare Dental, a modern family dental clinic in F-10 Markaz, Islamabad. The site must feel premium, medically professional yet approachable, fully responsive, and optimized for mobile booking conversions.
Brand Identity & Style:
Clinic Name: SmileCare Dental
Aesthetic: Clean, clinical trust meets warm accessibility. Medical precision without coldness. Think: white space, soft shadows, subtle gradients, rounded but not playful.
Primary Colors:
Primary: #0D7377 (Deep Teal — medical trust, modern)
Accent: #14A098 (Bright Teal — CTAs, highlights)
Neutral: #F5F5F5 (Soft Gray backgrounds)
Text: #1A1A2E (Near-black for readability)
White: #FFFFFF (Clean sections)
Typography:
Headlines: Inter or similar geometric sans-serif, bold weight (700)
Body: Inter regular (400) or system-ui
No serif fonts, no decorative scripts
I. Structural & Technical Requirements
Technology: Clean semantic HTML5, modern CSS (Flexbox/Grid, CSS variables), vanilla JS for interactions only where necessary.
Design System (CRITICAL — no generic buttons):
Buttons: Pill-shaped (border-radius: 50px), solid teal background, white text, subtle hover lift (transform: translateY(-2px)), no gradients, no shadows on default state, minimal shadow on hover.
Cards: White background, 1px solid #E5E5E5 border, border-radius: 16px, no heavy shadows (use subtle box-shadow: 0 4px 6px rgba(0,0,0,0.05)), clean padding.
Inputs: Same border-radius as buttons (50px), light gray border, focus state with teal border, no outline.
Spacing: Generous whitespace. Sections separated by 80-120px. No cramped layouts.
SEO & Speed: WebP images, lazy loading, semantic headings, alt tags.
Output: Separate files: index.html, styles.css, script.js
